Important FAQ?

What is the Document Object Model (DOM)?

The DOM represents a document with a logical tree. Each branch of the tree ends in a node, and each node contains objects.

How do you select an element from DOM?

The getElementById() method of the Document interface returns an Element object representing the element whose id property matches the specified string.

What is event delegation in the context of the DOM and why is it useful

event delegation allows you to attach a single listener to the parent container.

How do you manipulate an elements attributes and styles using the DOM?

If you want to change other attributes like id , href , or src , you can use the setAttribute() method.
